# Sistema de Controle de Ativos – Intranet

Sistema simples para controle, locação e devolução de ativos de uma intranet corporativa.

## Funcionalidades
- Cadastro de produtos
- Listagem em cards
- Locação de itens com:
  - Nome do colaborador
  - Matrícula
  - Data de devolução
- Devolução de itens
- Bloqueio de remoção de itens locados
- Persistência via LocalStorage

## Tecnologias
- HTML
- CSS (Flexbox)
- JavaScript (Vanilla)
